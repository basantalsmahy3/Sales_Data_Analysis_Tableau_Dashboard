# Sales Data Analysis – Interactive Tableau Dashboard

---

## Project Overview
This project develops an **interactive Tableau dashboard** to analyze and visualize sales data.  
The goal is to provide actionable insights on sales performance, trends, and regional distribution, using real datasets.

---

## Tools & Techniques
- **Tableau**: Interactive dashboards, charts, and maps  
- **Microsoft Excel / CSV**: Data preprocessing and exploration  
- **Data Visualization**: Bar charts, line charts, maps, and filters  

---

## Key Achievements
- Built bar charts showing sales by category  
- Created line charts displaying monthly trends  
- Developed maps visualizing sales by region  
- Added filters and interactive elements for dynamic exploration  
- Designed a professional and user-friendly dashboard highlighting most profitable regions  
- Delivered visual reports summarizing insights and findings
